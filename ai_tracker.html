NVIDIA 獨占 TSMC CoWoS 超過 60% 產能，AMD 僅能拿剩餘份額
封裝層
TSMC CoWoS 容量雖以 80% 年複合成長率擴建，但 NVIDIA 透過長期預訂鎖定了 60% 以上的 2025–2026 年產能。AMD 的 MI350/MI400 系列只能依靠剩餘分配，在市佔率上受到結構性限制。TSMC 也開始將部分封裝步驟外包給 ASE、Amkor 等 OSAT 廠商紓壓。
供應鏈影響鏈
TSMC（議價能力↑） → ASE / Amkor（訂單溢出↑） → AMD（市佔受限↓） → 非NVIDIA CSP 交貨期延長↓
SK Hynix 占 HBM 市場 62%；Micron 坦承只能滿足客戶需求的 50~66%
記憶體層
三家 HBM 供應商（SK Hynix、三星、Micron）都在上調合約價格，高個位數至高十幾%不等。Micron 甚至關閉了消費者品牌 Crucial，全力轉攻 HBM/AI 伺服器市場。HBM3E 每 GB 耗用的晶圓量是 DDR5 的 3 倍，導致傳統 DRAM 被排擠，PC/手機 DDR5 供給也跟著吃緊，HP、Lenovo 已提示漲價。
供應鏈影響鏈
SK Hynix / Micron（HBM ASP↑） → NVIDIA / CSP（GPU BOM成本↑） → 消費者 DDR5 漲價↓ → PC OEM（HP、Lenovo）毛利↓
TSMC 加速 CoWoS-L（LSI Bridge）技術，解決 Blackwell/Rubin 超越光刻極限問題
先進封裝
NVIDIA Blackwell 系列因晶片尺寸超越光刻機單次曝光極限（reticle limit），TSMC 被迫從 CoWoS-S（矽中介層）轉向 CoWoS-L（本地矽橋接），通過拼接多個晶片（chiplet）實現更大封裝面積。新建的嘉義 AP7 廠及亞利桑那兩座封裝廠均針對此設計，但美國製造的晶片目前仍需回台灣完成封裝。
供應鏈影響鏈
TSMC 嘉義廠（產能↑） → NVIDIA Rubin 量產時程提前↑ → 地緣風險集中在台灣⚠️ → Intel 封裝服務機會窗口↑

Microsoft 資本支出上修至 $1,900 億，$250 億直接源於記憶體元件漲價
需求端
微軟 CFO Amy Hood 在 Q1 財報電話會中透露，今年 CapEx 約有 $250 億是因元件成本（主要是記憶體晶片）價格飆升而增加，並非新增更多硬體容量。公司表示即使如此投入，「至少在 2026 年內 Azure 仍將持續處於容量受限狀態」。約 2/3 的支出流向 GPU/CPU，服務 Azure 客戶需求與 M365 Copilot。
供應鏈影響鏈
SK Hynix / Micron HBM（漲價成真↑） → Microsoft CapEx 膨脹但產能未增⚠️ → Azure 企業客戶等待時間拉長↓ → NVIDIA GPU 賣家議價能力↑
Google Cloud 積壓訂單單季從 $2,400 億暴增至 $4,620 億，Q1 雲端收入 +63%
需求端
Alphabet Q1 業績大超市場預期，Google Cloud 單季收入 $200 億，成長率 63% 創歷史新高，Gemini 3 企業版用戶季增 40%。Alphabet 隨之將全年 CapEx 指引上調 $50 億至 $1,800~$1,900 億。CFO 形容公司面臨「前所未有的企業和外部 AI 運算資源需求」。
供應鏈影響鏈
Gemini 3 企業採用↑ → TPU v7 + Trillium 晶片需求↑ → Broadcom（客製ASIC設計夥伴）↑ → TSMC CoWoS 需求進一步拉升↑
Meta 宣布同時部署 Broadcom 客製矽晶片 + AMD + NVIDIA，分散供應風險
採購策略
Meta CFO Susan Li 指出，CapEx 上調 $100 億至最高 $1,450 億中，部分反映元件漲價、部分是額外資料中心建設。更值得關注的是 Meta 宣布推出超過 1GW 的自研矽晶片（與 Broadcom 合作），並搭配大量 AMD 晶片補充 NVIDIA 系統。這是大型 CSP 明確降低 NVIDIA 單一依賴的最強訊號。
供應鏈影響鏈
Broadcom ASIC 業務↑ → AMD MI350/MI400 出貨增加↑ → NVIDIA 長期壟斷地位受挑戰↓ → TSMC：不管誰贏，封裝都找我⚠️

Boston Dynamics Atlas 在 Hyundai 喬治亞工廠展開首次真實場域測試
Physical AI
2026年1月4日 CBS《60分鐘》記錄了 Atlas 在現代汽車工廠的首次實地測試，標誌著人形機器人從實驗室正式進入工業場域。Figure AI、Boston Dynamics、Unitree 等廠商已開始向倉儲與製造客戶出貨商業級機器人。NVIDIA CEO Jensen Huang 在 GTC 2026 主題演講中描繪了「Physical AI 全棧路線圖」，以合成數據訓練機器人在真實環境中大規模運作。

Agentic AI：企業開始以「自然語言描述需求，AI 自動生成、測試並部署代碼」
Agentic AI
2026年早期，非技術領域專家已能透過純英文描述讓 AI 代理自主完成完整代碼庫的開發到部署。這大幅降低了軟體創作門檻，但也引發新的網路安全隱憂——AI 生成速度遠超過傳統安全審計節奏。「Vibe Coding」成為產業熱詞，企業對 AI 推論端算力（Inference Compute）需求快速攀升。
供應鏈影響鏈
推論晶片/邊緣 AI 需求↑ → 雲端 API 呼叫量（CSP 收入）↑ → 網路安全市場新需求↑ → 傳統軟體外包人力需求↓
NVIDIA 發布 Alpamayo：100億參數 VLA 自動駕駛模型，具備鏈式推理
Physical AI
Alpamayo 是 NVIDIA 推出的 Vision-Language-Action（VLA）模型，專為自動駕駛設計，整合了鏈式思維推理（Chain-of-Thought）以處理真實世界複雜駕駛場景。這是 Physical AI 從感知轉向「理解意圖、情境推理、自我糾錯」的關鍵技術跨越。它同時也驗證了 NVIDIA「AI Factory」平台——從數據飛輪到量產部署的全閉環商業模式。
供應鏈影響鏈
NVIDIA DRIVE 平台收入↑ → 車用 GPU/ORIN 晶片需求↑ → 車廠自動化採購 NVIDIA 平台↑ → 傳統 Tier-1 汽車供應商壓力↓
