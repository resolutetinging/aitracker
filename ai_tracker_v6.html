⚙️ 設定 GitHub Token
用於把分類設定儲存回 GitHub repo，只需設定一次。
前往 github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens
建立時選擇 aitracker repo → Contents: Read and write
Token 只存在你的瀏覽器，不會上傳到任何地方。
